# Simple To-do App


## Functionality

This simple web application contains a list of initial "checkable" and "deletable" tasks with optional due dates, along with the ability for user-added tasks through a form (accessible via button). The project has been updated to include a dynamic counter at the top showing total completed and total overall tasks.

## Technology/Techniques

* JavaScript form validation via OOP with classes
* Appearing/disappearing modal with form for user-added tasks
* Exporting/importing for use of relevant variables and functions throughout file structure
* Responsive design for smaller computers, tablets, and smartphones
* Flat BEM file structure
* Class inheritance with loosely coupled components
